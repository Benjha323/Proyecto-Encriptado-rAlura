# Encriptador de Texto

Este proyecto es una aplicación web simple que permite encriptar y desencriptar texto utilizando un conjunto de reglas específicas.

## Funcionalidades

- **Encriptar texto**: Convierte el texto ingresado en una versión encriptada siguiendo un conjunto de reglas predefinidas.
- **Desencriptar texto**: Convierte el texto encriptado de vuelta a su forma original.
- **Copiar texto**: Permite copiar el texto encriptado/desencriptado al portapapeles.

## Tecnologías Utilizadas

- **HTML5**: Estructura del contenido web.
- **CSS3**: Estilización y diseño responsivo.
- **JavaScript**: Funcionalidades de encriptado, desencriptado y copia.

## Estructura del Proyecto

- **index.html**: Contiene la estructura básica de la página web.
- **style.css**: Archivo de estilos que controla el diseño y la responsividad de la página.
- **script.js**: Contiene las funciones de encriptado, desencriptado y copiado.

## Responsividad

El diseño está optimizado para diferentes rangos de pantalla:

- **Móviles**: Entre 320px y 480px.
- **Tablets**: Entre 768px y 1024px.
- **Pantallas grandes**: Más de 1200px.
